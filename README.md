# Todo App (AWS Serverless)

A fully serverless Todo App built using AWS services.

## Live Demo
👉 https://d1xyge1oopbt60.cloudfront.net

##  Tech Stack
- AWS S3 (Frontend Hosting)
- AWS CloudFront (CDN + HTTPS)
- AWS Lambda (Backend)
- AWS API Gateway
- AWS DynamoDB (Database)

##  Features
- Add tasks
- View tasks
- Delete tasks
- Responsive UI
- Fast global access via CloudFront

##  Project Structure
- index.html → Main UI
- style.css → Styling
- script.js → Frontend logic

## How It Works
1. User interacts with UI
2. API Gateway triggers Lambda
3. Lambda stores/fetches data from DynamoDB
4. Data reflects on UI
